# Hospital Management System

A professional, enterprise-level Hospital Management System built with modern technologies for real-world healthcare operations.

## 🏥 Overview

A complete hospital software solution designed for hospitals, clinics, medical centers, and healthcare organizations. This system provides comprehensive management for patients, doctors, appointments, billing, pharmacy, laboratory, and more.

## 🛠️ Technology Stack

### Frontend
- **HTML5** - Semantic markup
- **Tailwind CSS** - Modern responsive design
- **JavaScript** - Dynamic functionality
- **Chart.js** - Analytics and reporting

### Backend
- **C++** - High-performance server
- **Crow Framework** - Lightweight HTTP framework
- **SQLite** - Database management
